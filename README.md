# Mental-Health-Tech-Analysis
End-to-end Data Analysis Project on Mental Health Trends in the Technology Industry using Python.

# Mental Health in Tech Industry Analysis

## Project Overview

This project analyzes mental health trends among technology professionals and identifies factors affecting treatment and workplace well-being.

## Dataset

Mental Health in Tech Survey Dataset

## Objectives

- Data Cleaning
- Exploratory Data Analysis
- Visualization
- Insight Generation
- Recommendations

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Analysis
4. Visualization
5. Insights
6. Recommendations

## Key Findings

- Family history impacts treatment decisions.
- Workplace support improves mental health outcomes.
- Younger employees report more mental health concerns.
